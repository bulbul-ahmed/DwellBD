# Owner Dashboard – Flat Rent/Sell Platform (Dhaka)

## Purpose
The Owner Dashboard enables flat owners to list properties, manage inquiries, schedule visits, close rental/sale deals, and manage tenants efficiently. The dashboard is mobile-first, simple, and aligned with real rental behavior in Dhaka.

---

## 1. Core Principles
- Mobile-first design
- Bengali & English support
- Minimal steps to close a deal
- Strong trust & verification signals
- Optimized for offline visits and phone calls

---

## 2. Dashboard Navigation Structure

1. Overview
2. My Flats
3. Inquiries
4. Visit Requests

## 3. Feature Breakdown

## 3.1 Overview
High-level summary for owners.

### Features:
- Total flats listed
- Available / Rented / Booked count
- New inquiries today
- Upcoming visits
- Recent activity feed

### Metrics:
- Flat views (daily / weekly)
- Inquiry count per flat

---

## 3.2 My Flats (Property Management)

### Features:
- Add new flat
- Edit flat details
- Pause / Activate listing
- Delete listing
- Upload & manage photos

### Flat Status:
- Available
- Under Discussion
- Booked
- Rented

### Flat Information:
- Location (area + map pin)
- Bedrooms / Bathrooms
- Size (sqft)
- Rent
- Advance amount
- Service charge
- Gas type
- Lift / Parking / Generator
- Tenant preference (Family / Bachelor / Office)
- Availability date

---

## 3.3 Inquiries (Lead Management)

### Features:
- Tenant inquiry list
- Masked phone number
- Chat or call via platform
- Tenant type & budget
- Inquiry timestamp

### Actions:
- Accept inquiry
- Reject inquiry
- Mark as interested
- Mark as not suitable
- Mark as deal closed

---

## 3.4 Visit Requests (Scheduling)

### Features:
- Visit request list
- Date & time selection
- Approve / Reschedule / Reject visit
- Simple calendar or list view
- Visit reminders (SMS / in-app)

### Notes:
- Owner notes per visit
- Tenant suitability comments

---

## 3.5 Deals (Closing the Rental / Sale)

### Features:
- Confirm tenant
- Finalize rent & advance
- Set move-in date
- Generate rental agreement
- Upload signed agreement (PDF / image)

### Payment Tracking:
- Payment method:
  - Online via platform
  - Offline (manual confirmation)
- Advance received status
- First month rent received status

---

## 3.6 Tenants (Post-Rent Management)

### Features:
- Tenant profile
- Move-in date
- Rent amount
- Payment due date
- Notice period
- Tenant status (Active / Vacated)

---

## 3.7 Payments & Rent Tracking

### Features:
- Monthly rent history
- Payment status (Paid / Due / Late)
- Manual payment marking
- Automated rent reminders (SMS / WhatsApp)
